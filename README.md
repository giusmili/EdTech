# La Grande Classe

Application migree vers Next.js avec App Router, en conservant la structure visuelle et la charte graphique existantes.

## Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- Motion
- Lucide React

## Lancer le projet

1. Installer les dependances :
   `npm install`
2. Lancer le serveur de developpement :
   `npm run dev`
3. Generer le build de production :
   `npm run build`
